# autosetup-scripts
autosetup-scripts

- for nix pkgs and nativefier webapps you might add files on ``~/.local/share/applications``
- ex:

lichess.desktop
```
[Desktop Entry]
Version=1.0
Type=Application
Exec=/home/arega/Documentos/Nativefier/Lichess-linux-x64/Lichess
Name=Lichess
Comment=Lichess.org online chess
```
